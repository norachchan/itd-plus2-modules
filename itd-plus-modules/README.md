# ITD+ — Script for итд.com

ITD+ is a functional extension designed to enhance the user interface and add hidden features to the itd.com platform. This project is created for those who want to get the most out of the site's functionality.

#itdStatus #itdPlus

---

### Key Features
The mod expands standard functionality by adding:
* Enhanced Visualization: Custom profile banners (images and GIFs) and verification badges for all users.
* Privacy and Control: Private profile features and an advanced Blacklist to filter the feed.
* Informational Tools: Display of full registration dates and precise last-seen online status.
* Interactivity: Clickable links and mentions.
